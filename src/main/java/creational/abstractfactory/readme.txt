
AbstractFactory

Назначение: предоставляет интерфейс для создания семейств взаимосвязанных и взаимозависимых объектов,
            не указывая их конкретных классов.

Применимость:   система не должна зависеть от того, как создаются, компонуются и представляются входящие в нее объекты.

Плюсы:  Изолирует конкретные классы.
        упрощает замену семейств продуктов
        гарантирует сочетаемость продуктов
Минусы: Трудно поддержать новый вид продукта.
